# vue-questionnaire

> 简易问卷系统,目前功能正在逐步完善当中...

## 安装

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

## API

## 路由

`/login` 登录

`/platform/list` 问卷列表

`/platform/new` 新建问卷

`/platform/new/edit` 编辑问卷

`/view` 问卷展示
